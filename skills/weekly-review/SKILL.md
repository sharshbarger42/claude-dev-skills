## Instructions

### Step 1: Review the week
- Look at ROUTINES.md for this week's daily routine tracking
- Calculate completion rates for each routine item
- Note any streaks or patterns (improving, declining, consistent)

### Step 2: Review weekly goals
- Check WEEK.md for this week's goals
- Mark what was completed vs carried over
- Note any accomplishments not in the original plan

### Step 3: Check next week's calendar
- Use the Google MCP server to fetch calendar events for the next 7 days
- List any appointments, deadlines, or events coming up
- Flag anything that needs preparation

### Step 4: Weekly routine items
- Check off completed weekly routine items (groceries, meal plan, cleaning, etc.)
- Note what still needs doing before the week ends

### Step 5: Set up next week
- Create a fresh WEEK.md for the upcoming week
- Carry over any unfinished goals that are still relevant
- Suggest 2-3 focus areas based on patterns from Step 1
- Include the calendar events from Step 3

### Step 6: Weekly summary
Present a concise briefing:
1. Routine completion rates (daily items)
2. Weekly goals: done vs carried over
3. Wins for the week
4. Areas to improve
5. Next week's calendar highlights
6. Suggested focus for next week

Keep it honest and concise. No fluff.
